# yiman

yiman是一款使用 MVP + Retrofit2 + Rxjava + Event 等框架开发的阅读软件。
包括新闻、图片、段子视频模块，项目的大部分功能已经完成。

## 目的
初学者一起探讨学习，对性能优化、代码编写尽可能标准、朝标准APP不断完善改进
